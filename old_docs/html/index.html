py4t - 教青少年寫python¶

py4t是多個python套件的整合，目的是要搭一座學習的橋，從Scratch積木到Python文字，教青少年學習程式設計。

【安裝檔下載】

  • thonnyPy4t 安裝檔(for windows)

【教學模組範例】

  • 海龜幾何繪圖
  • 物理碰撞模擬
  • 3D模擬
  • 視覺影像
