## 階段2: 簡單轉動


🎦 範例影片

: 建構中

📄 Py4t程式碼

```python
from 模擬3D模組 import *

旋轉中心 = 新增物體()

for z in [-1, 0, 1] :
    for y in [-1, 0, 1] :
        for x in [-1, 0, 1] :            
            物體 = 新增6面貼圖方塊()
            物體.材質貼圖 = '魔方6面.png'
            物體.位置 = [x,y,z]
            if y == 1:
                物體.親代 = 旋轉中心

def 當更新時(dt):
    旋轉中心.旋轉y += 1
    
模擬主迴圈()
```

: 註: 材質貼圖檔需匯入並修改。
