???+ example "範例程式 海龜畫多邊形 - - - - - - - (海龜畫多邊形 1/2 新檔)"

    === "💻程式碼"

        ```python
        from 海龜模組 import *
        速度('fastest')

        for 數 in range(500) :
            向前(數*2)
            右轉(360/5 - 1)    

        完成()
        ```

: ![turtle_star](turtle_star.jpg)

: 接下來，利用官網上的說明，將程式改寫為英文程式。

<BR/>
<BR/>

???+ example "範例程式 turtle英文程式 - - - - - - - (海龜畫多邊形 2/2 接續)"

    === "💻程式碼"

        ```python
        from turtle import *
        speed('fastest')

        for i in range(500) :
            forward(i*2)
            right(360/5 - 1)    

        done()
        ```
