視覺範例 電視牆 
================================

在每一張取影的攝影機影像中，先將影像轉成灰階，並把影像縮小(150)，再利用多維陣列的拼貼功能，在直向與橫向兩個方向中，各拼貼4次，就做出了如同電視牆的效果。

註：電腦上需要有視訊攝影機
