視覺影像¶

這個學習模組是建立在opencv-python的電腦視覺函式庫上。從攝影機擷取到的影像，取得numpy多維陣列的影像，可以對影像做出改變亮度、對比、縮放、負片和左右翻轉等效果，並進行電腦影像的處理。

註：電腦上需要有視訊攝影機
